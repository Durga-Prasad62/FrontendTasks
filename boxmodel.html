  NodeJS


Node.js is an open-source, cross-platform JavaScript
     runtime environment that executes JavaScript 
     code outside of a web browser. It is built on 
     Chrome's V8 JavaScript engine and allows 
     developers to use JavaScript for 
     server-side and networking applications.
